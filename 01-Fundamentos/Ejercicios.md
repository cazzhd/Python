## 1. ¿Cuál es la salida del algoritmo?
```py
n = 4; k = 2

print (n)
print (-n)
print (n-)
print (n + k)
print (k)
print (n, k)
print ("")
print ("n%3.2g" % n)
print ("n * n = %3.2g" % n*n)
print (n * n)
print ('n')
```

Produciría la siguiente salida:
```txt
4
-4
ERROR
6
2
4, 2

n 4
n * n =   4 n * n =   4 n * n =   4 n * n =   4
16
n
```
